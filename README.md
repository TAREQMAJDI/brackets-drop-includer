# brackets-drop-includer

## How to use
Please drag and drop some files into the modal window, and this extension inserts tags.

(link tag, img tag, script tag, or php include statement)

## Shortcut Key
### Open The Modal Window (drag and drop zone)
Ctrl(Cmd) + .

### Show an "Open" dialog
#### Root Path
Ctrl(Cmd) + Shift + .

#### Relative Path
Ctrl(Cmd) + Shift + Alt + .
